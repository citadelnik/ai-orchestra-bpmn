📧 Этап 1: Получение и валидация заявки (2-5 минут)

  • Мониторинг почты: Автоматическое сканирование входящих писем
  • Парсинг данных: Извлечение URL сайта, списка продуктов, контактов
  • Валидация: Проверка корректности URL, доступности сайта
  • Уведомление: Подтверждение получения заявки клиенту
  • Создание задачи: Инициация процесса анализа в системе

🎯 Этап 2: Оркестрация и запуск агентов (1-2 минуты)

  • Инициализация: Загрузка конфигурации для каждого агента
  • Распределение задач: Назначение специфических целей каждому агенту
  • Запуск параллельного выполнения: Одновременная работа всех агентов
  • Мониторинг: Отслеживание прогресса и обработка ошибок

🤖 Этап 3: Параллельная работа агентов (30-40 минут)

  • Scout Agent: Сбор SEO данных, анализ конкурентов, техаудит сайта
  • Analyst Agent: Расчет traffic_value, прогноз конверсий, ROI анализ
  • Strategist Agent: Создание roadmap, определение ICP, планирование
  • Синхронизация: Обмен данными между агентами при необходимости

📊 Этап 4: Формирование и доставка отчета (5-15 минут)

  • Агрегация данных: Сбор результатов от всех агентов
  • Применение шаблона: Форматирование в Markdown структуру
  • Валидация отчета: Проверка полноты и корректности данных
  • Доставка: Отправка готового отчета на email клиента
  • Архивирование: Сохранение результатов для аналитики
